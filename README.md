# Mi Viaje APP

Aplicación móvil desarrollada como proyecto académico utilizando Ionic Framework, orientada a conectar pasajeros y conductores para la realización de viajes en automóvil, de forma similar a plataformas de transporte colaborativo.

El sistema permite a los usuarios solicitar viajes indicando punto de inicio y destino, mientras que los conductores pueden visualizar solicitudes disponibles y seleccionar los viajes que desean realizar.

## Funcionalidades principales

### Pasajeros
- Registro e inicio de sesión
- Solicitud de viajes indicando origen y destino
- Visualización del estado del viaje
- Pago del servicio al finalizar el trayecto

### Conductores
- Visualización de viajes disponibles
- Información del pasajero y cantidad de personas
- Selección de viajes a realizar
- Definición del valor del servicio

## Tecnologías utilizadas
- Ionic Framework
- Android
- TypeScript
- HTML y CSS
- Integración de lógica de negocio para gestión de viajes

## Mi rol en el proyecto
- Desarrollo de la aplicación móvil
- Implementación de funcionalidades para pasajeros y conductores
- Diseño de flujos de uso y lógica del sistema
- Integración de procesos de solicitud y finalización de viajes

## Objetivo del proyecto
Desarrollar una aplicación móvil funcional que simule el funcionamiento de una plataforma de transporte colaborativo, aplicando conceptos de desarrollo móvil, experiencia de usuario y lógica de negocio.
